Tween 动画函数封装
---------

1、理解在时间范围内位置的变化

> 即 CSS 中的 animation-timing-function ，可以理解为通过函数 y = f(x) 得出的 y 的变化曲线。

2、写一个能够执行数学函数的通用的 js 动画函数

3、优化：能够同时执行多个动画，每个动画执行不同的数学函数

4、优化：能够传入自定义时间函数，或者自定义如三阶贝塞尔曲线的参考点来执行动画
